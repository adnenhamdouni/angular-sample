Create powerful animations based on the Web animations API

You must import animations.

New to angular 4, animations must be imported manually, as they are no longer a part of the Angular core library. To do this, just use nmp command:

How to configure and install animations?

npm install @angular/animations@latest --save

Then on app module ts, import BrowserAnimationsModule from '@angular/platform-browser/animations'

and add it to imports into @ngModule

How use it?

on your component, we need to import trigger,state,style,transition,animate,keyframes from '@angular/animations'

then add animations to @component
